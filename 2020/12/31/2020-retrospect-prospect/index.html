2012 年度總結

2020 年真的是變化多端的一年，今年也是有滿滿的收穫，期許自己一年比一年更精彩，訂下目標持續前進。

工作方面

今年中我在公司經歷組織換組，榮幸與兩位社群大大一起 Kakashi 與 Ｃhamp 網路黑貓 一起合作交流 Infrastructure/Could Project，雖然知道團隊初期需要解決很多臨時問題與釐清目標，但每次討論過程都學到很多思維與技術，自己覺得非常充實，並在年末把大部分繁瑣事務規格化，完善成日常　CI/CD　流程，終於可以安心喝咖啡。

至於剛剛好今年 COSCUP 2020 Conference，我們 Infra 小隊都有參加，只有我今年沒有去當講者，活動當天主要都是幫忙　CNTUG　社群議程軌。

兩場都抽空去玲聽

目前在公司與閒暇研究 Kubernetes/Singularity Container/Jenkins，今年也寫幾篇筆記文章記錄，但一直沒甚麼題目方向，沒有達成去年訂的寫文章目標。

社群方面 (Cloud Native Taiwan User Group)

去年開始正式協助社群並擔任社群金流負責人，原本自己從協助年會攝影志工到社群負責人轉變，今年算是更能完整理解籌備社群活動背後是如何去運作。

從每個月 Meetup 宣傳，需要從講者招募、場地協商、線下擺攤推廣社群、捐款請款流程，體會社群夥伴為社群無私付出的熱情。最後 12 月的「Cloud Native Taiwan 年末聚會」，是今年 Meetup 最大的一場，有 RedHat 贊助議程場面超豪華。

#COSCUP CNTUG 社群議程軌:

一年一度的 COSCUP，感謝 pohsienshih​與 Phil Huang 小飛機 這次擔任社群議程軌負責人，回顧籌備過程聽到很有感觸的一段話：「任何的技術分享對於許多正在學習技術的人員來說，都是很好的學習以及交流的機會」，附上協助　COSUCP X CNTUG 議程記錄。


#Kubernetes Summit 2020 大會協助擺攤

#籌備年末聚會

從零開始籌備年末大聚會，雖然時程與安排上非常趕，也很感謝主要這次負責 Ｈazel辛苦聯繫與安排流程，這次活動也終於讓社群有收入，明年要更早確認所有事物，這次活動算是每年年末認親與交流大會，現場感受超棒！

Lightning Talk

今年因為沒什麼題目能分享，還是想透這個機上去講一場 Lightning Talk - 「從零開始導入 Singularity Container 歷程雜談」，算是給自己一個總結也很榮幸今年的自己能參與協助社群。

今年給自己很多深度體驗，找回生活平衡

  • 利用公司 Coffe Time 時間鑽研喜愛的手沖豆品
  • 年中決定要減體酯，實施16/8
  • 完成一季不缺席，每週的飛輪課(風雨無阻)
  • 買了第一台空拍機　(DJI Mavic Mini)，轉換自己攝影視角
  • 被同事列了一系列的 To Buy List 只完成一項（空拍機）
  • 自己耗時組了一台桌機，HomeLab＋打game無煩惱了
  • 深度體驗：攀岩(抱石)、深潛水、滑Sup、搭蹦蹦車、
  • 開始經營自己攝影 IG(@yylin_photography)
  • 寫了一篇 2020攝影回顧

2021 展望

  • 目標多撰寫技術文章，找Meng/Pohsienshih夥伴一起經營　Medium。
  • 增加自己演講機會，積極參與　Conference 或是 Meetup 分享
  • 貢獻有興趣的開源專案
  • 多看點書，淘汰實體書存放Kobo電子書

結語

今年年中開始算是一個大轉變，也很高興工作跟生活有慢慢找到平衡，平時時間管理還是要重新拿捏，從健身與飲食開始積極調整每一天的生活，另外就是社群參與並認識很多同領域的技術好朋友，我想新的一年希望自己思維上有更多的成長，技術能力能更加扎實繼續前進，最後感謝2020年幫助過我的每個人。
